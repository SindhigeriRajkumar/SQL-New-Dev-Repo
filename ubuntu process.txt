How To Install Ubuntu in Windows 11 Using Subsystem For Linux

power shell -->run as adminstrator


Enable WSL 1:

paste 1:    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
------->



Enable WSL 2:

paste 2:   dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
------->

Set WSL 2 as the default WSL environment:

after restart:
--------------

wsl.exe --set-default-version 2
